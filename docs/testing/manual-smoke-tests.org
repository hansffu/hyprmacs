#+title: hyprmacs Manual Smoke Tests
#+date: 2026-04-15

* Task 1 Bootstrap
** Emacs package
- Run:
  =emacs -Q --batch -L emacs -L tests/emacs -l bootstrap-tests.el -f ert-run-tests-batch-and-exit=
- Expect:
  - both bootstrap ERT tests pass
  - =hyprmacs.el= loads successfully

** Plugin bootstrap build
- Run:
  =nix build .#hyprmacs-plugin=
- Expect:
  - Nix build succeeds
  - a plugin shared object is available under =result/lib/*.so=

** Nested Hyprland smoke test
- Start the nested session:
  =just run=
- Inside the nested session, run:
  =just load=
- Expect:
  - Hyprland loads the plugin from the Nix store without crashing
  - the plugin can be unloaded cleanly with:
    =hyprctl plugin unload /nix/store/.../lib/<plugin>.so=

* Notes
- The agent sandbox may not expose Hyprland headers directly even when the flake does in the user environment.
- If the plugin binary is built in fallback mode, do not load it. Rebuild through =nix build .#hyprmacs-plugin= in the user environment first.
